# Exported Functions

This library exports the following functions:

- **`mysql(host, port, user, password [, db])`**:
  
  Returns a new instance of the Mysql object.

  - **Parameters:**
    - *string* `host`
    - *number* `port`
    - *string* `username`
    - *string* `password`
    - *string* `database` (Optional)
  - **Returns:** `Mysql`
